# corpora-extract

Script to convert corpora to input as provided to beacon-fuzz harnesses.

Useful for debugging, as the harnesses receive input after pre-processing.

## Getting started

Running it then should be as simple as:

```console
$ make
$ ./bin/corpora-extract
```

### Testing

``make test``
